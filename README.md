**Scientific Calculator in C**

**Project Overview:**

The Scientific Calculator is a menu-driven console application developed in C that performs both basic arithmetic and advanced mathematical calculations. The program allows users to perform operations such as addition, subtraction, multiplication, division, exponentiation, square roots, trigonometric functions, and logarithmic calculations through an interactive user interface.

This project demonstrates the practical implementation of mathematical functions using C's standard libraries and helps strengthen concepts such as control structures, loops, conditional statements, and function-based programming.


**Objectives:**

Develop a menu-driven calculator application in C.

Perform both basic and scientific mathematical operations.

Utilize the math.h library for advanced calculations.

Enhance understanding of user input handling and program flow control.

Build a practical application that demonstrates real-world usage of C programming.
